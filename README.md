## Getting Started

This project use [MetaboHUB's Latex template project](https://services.pfem.clermont.inrae.fr/gitlab/metabohub/mth_template).

### Prerequisites

- Latex compiler
- Git
- Make
- Docker (optional)

```
sudo apt-get install \
    texlive-full \
    git \
    make
```
